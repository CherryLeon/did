# 实证研究规范

> 本文档定义论文实证研究的标准操作流程、数据要求和质量标准。

## 1. 核心实证方法

本项目实证研究采用**双重差分法（DID）** 作为主分析方法，配套使用**倾向得分匹配（PSM）**、**平行趋势检验**和**多维稳健性检验**。

## 2. 双重差分法（DID）规范

### 2.1 标准 DID 模型

```python
# 基准 DID 模型
# Y_it = α + β·Treat_i × Post_t + γ_i + δ_t + ε_it

# 其中：
# Y_it: 被解释变量（客户经营成效综合指数 CEMI）
# Treat_i: 处理组虚拟变量（1=处理组，0=对照组）
# Post_t: 政策后虚拟变量（1=政策实施后，0=政策实施前）
# γ_i: 个体固定效应
# δ_t: 时间固定效应
# β: 核心关注的处理效应系数
```

### 2.2 准自然实验构造要求

| 要求 | 说明 | 检验方法 |
|------|------|----------|
| 外生性 | 处理/对照组划分基于外生事件，非人为设定 | 描述性分析 |
| 可比性 | 两组在政策前特征应高度可比 | PSM平衡性检验 |
| 独立性 | 控制组不受处理影响 | 理论论证 |

### 2.3 样本量要求

- 论文要求：处理组 ≥ 30 个单位，对照组 ≥ 30 个单位
- 观测值数量应满足统计功效分析要求（Power ≥ 0.8）
- 面板数据应说明观测期长度和观测值总量

## 3. 倾向得分匹配（PSM）规范

### 3.1 PSM 实施步骤

```
1. 选择协变量：基于理论选择影响处理分配的混淆变量
2. 估计倾向得分：Logit/Probit模型 P(Treat=1|X)
3. 进行匹配：核匹配/最近邻匹配/卡尺匹配
4. 平衡性检验：匹配后标准化偏差 < 10%（理想 < 5%）
5. 共同支撑检验：处理组与对照组倾向得分分布重叠
```

### 3.2 协变量选择原则

- 与处理分配相关
- 与被解释变量相关
- 不受处理影响（不位于处理到结果的因果路径上）
- 政策前可观测

### 3.3 平衡性检验标准

| 指标 | 可接受标准 | 理想标准 |
|------|-----------|----------|
| 标准化偏差 | < 10% | < 5% |
| t检验 p值 | > 0.05 | > 0.1 |
| 联合F检验 | 不显著 | - |

## 4. 平行趋势检验规范

### 4.1 事件研究法

```python
# 事件研究法 DID 模型
# Y_it = α + Σ(k=-K,k≠0) β_k · D_{i,k} + γ_i + δ_t + ε_it

# D_{i,k}: 处理前k期(k<0)或处理后k期(k>0)的交互项
# β_k: 处理前各期系数应不显著（支持平行趋势）
#      处理后各期系数应逐渐显著（支持动态效应）
```

### 4.2 平行趋势检验要求

- 必须报告政策前至少 3 期（理想 6 期）的交互项系数
- 政策前系数联合 F 检验应不显著（p > 0.05）
- 需附可视化图表展示政策前后的效应演变

## 5. 稳健性检验体系

### 5.1 必须执行的稳健性检验

| 检验类型 | 方法 | 通过标准 |
|----------|------|----------|
| 安慰剂检验 | 随机虚构处理组 1000 次 | 真实系数位于伪系数分布 2.5% 尾部之外 |
| 排除政策干扰 | 控制其他同时发生的政策变化 | 核心系数保持稳健 |
| 样本调整 | 剔除特殊样本/收窄时间窗口 | 核心系数保持稳健 |
| PSM+Rosenbaum界限 | 评估不可观测混淆的敏感性 | Γ ≥ 1.5 时结论仍稳健 |

### 5.2 显著性水平

| 统计量 | 显著性水平 |
|--------|-----------|
| * | p < 0.1 |
| ** | p < 0.05 |
| *** | p < 0.01 |

## 6. 经济价值折算

### 6.1 折算方法分类

| 类型 | 说明 | 示例 |
|------|------|------|
| 因果推断折算 | 直接基于 DID 系数计算 | 转化率提升 × 客户基数 |
| 辅助估算 | 描述性统计支撑 | 效率提升 × 单位成本 |

### 6.2 报告要求

- 必须明确区分因果推断类和经济估算类
- 提供置信区间（理想 95% CI）
- 折算假设应透明化

## 7. 变量命名规范

```
CEMI: 客户经营成效综合指数（Customer Management Effectiveness Index）
Treat: 处理组虚拟变量
Post: 政策后虚拟变量
ATT: 平均处理效应（Average Treatment Effect）
PSM: 倾向得分匹配（Propensity Score Matching）
DID: 双重差分（Difference-in-Differences）
```
